One-time setup

This app uses Google Apps Script (free, no extra accounts needed) to save sign-ups. Only the organizer needs to do this once.

  1. Go to sheets.google.com and create a new blank spreadsheet. Name it anything.
  2. Click Extensions → Apps Script in the menu bar.
  3. Delete everything in the editor. Paste in the contents of the Code.gs file. Click Save (floppy disk icon).
  4. Click Deploy → New deployment. Click the gear icon next to "Select type" and choose Web app.
  5. Set Execute as: Me and Who has access: Anyone. Click Deploy. Approve any permissions.
  6. Copy the Web app URL (starts with https://script.google.com/macros/s/...) and paste it below.

Saved in your browser — only needs to be entered once per device.
